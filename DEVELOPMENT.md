# Backend Project Generator

## 🛠️ Development

### 1. Install Dependencies

```bash
npm install
```

### 2. Run in Development Mode

```bash
npm link
backend-generator
```

### 3. Run Tests

```bash
npm test
```

### 4. Contributing

To contribute to the project, please read the [CONTRIBUTING.md](./CONTRIBUTING.md) file.
